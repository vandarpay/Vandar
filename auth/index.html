مرحله اول: ارسال اطلاعات و دریافت توکن

برای فراخوانی دیگر متدهای وندار نیاز است که در اولین قدم دریافت توکن را فراخوانی کرد، همچنین با توجه به عمر پنج روزه‌ی هر توکن توصیه می‌شود هر چهار روز یک بار دریافت توکن مجددا فراخوانی گردد. با فراخوانی این سرویس اطلاعات کاربری و توکن مورد نیاز برای استفاده در هدر دیگر سرویس‌ها، بازگردانده می‌شود.

METHOD: post
URL: https://api.vandar.io/v2/login

پارامترهای مجاز

Name Type Status
mobile string required
password string required

توضیحات

  • mobile (اجباری): شماره موبایل ثبت شده در وندار.
  • password (اجباری): رمز عبور حساب کاربری وندار.

نمونه json 

{
    "mobile": "09123456789",
    "password": Very-strong-pa$$w0rd
}
